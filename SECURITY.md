# 🛡️ SECURITY POLICY

## Supported Versions
Currently, only the main branch (v2.6.x) receives active security updates.

## Reporting a Vulnerability
If you find a security flaw in the **ARCHITECT CG-223** logic:
1. **DO NOT** create a public GitHub Issue.
2. Contact the Executive Lead via **Discord** (Eagle Community) or **Telegram** (@YourUsername).
3. We aim to respond within 48 hours to secure the Bamako Node.

## Safe Handling
* Never commit your `.env` file.
* Rotate your `TELEGRAM_TOKEN` if it is ever exposed in logs.
